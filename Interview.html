HTML Interview Questions

Q1. Difference between div and span ?

Ans. Div- Div is a blocl-level element and related to page section. You have to define an empty div tag and attach a class attribute to it in the Html.

Span- The tag is an inline container used to mark a part of text. The tag is easily styled by CSS.The tag is much like the element, but this is a inline element.


Q2. Difference between class and IDs ?

Ans. Class- The class attribute specifies one or more classnames for an element.It can also be used by a Javascript.

IDs- ID is used to identify one element in html.It is an unique element.

Q4. Difference between block and inline ?

Ans. Block level element: It has begin with a new line.It will be extend left to right. Example: div, p, ol, dl.

Inline level element: Inline elements display in a line.An anchor is an example of a inline element.Example: a, img, input.
